# OpenPokeDex

OpenPokeDex is a minimal, fast, and open-source Pokédex application.  
Built with [React](https://react.dev/), [Vite](https://vitejs.dev/), and [PokéAPI](https://pokeapi.co/), it provides a smooth browsing experience for Pokémon species, complete with flexible pagination and responsive design.

## Features

- 📚 Browse Pokémon with real-time data from PokéAPI.
- 🔍 Detailed individual Pokémon pages.
- 📄 Pagination for easy navigation.
- ⚡ Fast loading and lightweight architecture.
- 📱 Mobile-friendly and responsive layout.
- 🛡️ Licensed under the MIT License.

## Technologies Used

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [React Router](https://reactrouter.com/)
- [PokéAPI](https://pokeapi.co/)
